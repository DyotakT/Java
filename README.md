# Spiral Square

This program creates a square of given side with numbers spiralling towards the center.

Sample output:

```
Enter the size of the side of the Spiral Square: 5


The Spiral Square is: 

1	2	3	4	5	


16	17	18	19	6	


15	24	25	20	7	


14	23	22	21	8	


13	12	11	10	9	
```

## Instructions:

* Clone this repo:
```bash
$ git clone https://github.com/DyotakT/Java.git
```
* Compile:
```bash
$ cd Java
$ javac Spiral.java
```

* Run:
```bash
$ java Spiral
```
